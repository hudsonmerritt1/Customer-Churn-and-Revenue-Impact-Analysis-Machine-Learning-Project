This project uses machine learning and business intelligence techniques to predict customer churn and estimate the financial impact of customer loss using the Telco Customer Churn dataset. Multiple models were developed and compared, including logistic regression, decision trees, neural networks, and a random forest ensemble model. The project focused on identifying customers most likely to churn while also calculating Expected Revenue At Risk by combining churn probability with estimated future revenue loss. Key findings showed that customers with month-to-month contracts, shorter tenure, and fiber optic internet service were more likely to churn, while long-term contracts and longer tenure reduced churn risk. The project demonstrates how machine learning can be applied in a business context to support customer retention strategies, reduce revenue loss, and improve data-driven decision-making. Technologies used include Python, Pandas, Matplotlib, and Scikit-learn.
